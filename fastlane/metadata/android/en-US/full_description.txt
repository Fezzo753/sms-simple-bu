SimpleBackup is a one-tap backup app for SMS, MMS text bodies, and call-log entries from selected contacts on your device.

The output is a single self-contained HTML file (with a JSON sibling) saved to the app's private storage. The HTML opens in any browser — desktop or phone — and includes search, filters by date / direction / kind, multi-contact selection, Print to PDF, and Export to TXT. Subsequent runs merge new events into the same file.

Everything runs entirely on-device. No network access, no analytics, no cloud sync, no accounts. The app does not even request the INTERNET permission.

<b>Permissions</b>

* Read SMS — required to capture text messages and MMS bodies.
* Read call log — required to capture call entries.
* Read contacts — used so backups display contact names instead of bare numbers.

<b>What it does</b>

* Pick contacts (or "include all").
* Tap "Back up now". Phases: SMS → MMS → calls → merge → HTML.
* View the result in-app via WebView, or share the .html file to anywhere.
* Re-run later: new events merge into the existing file (no duplicates).

<b>What it does not do</b>

* No restore. The exported JSON is structured so a future tool could restore, but this app only captures.
* MMS image / video media is captured as a "[image]" or "[video]" placeholder in v0.1; only text bodies survive.
* Sideload only. Not on Play Store. Google Play Protect may warn at install — this is normal for SMS-capable apps outside Play; tap "Install anyway" or "More details" → "Install".

<b>Why a separate app</b>

The popular alternative — SMS Backup &amp; Restore — captures everything on the device. SimpleBackup is for the case where you only want a paper-trail-quality export for a small set of contacts, in a format that opens in any browser without needing the original app installed.
